自用poe做装小程序

使用说明：
1、运行该程序后按下"<alt>+2", 获取通货、装备对比的坐标。并将坐标填入config.ini中
2、根据流亡编年史中词缀补充做装物品的所有前缀和后缀，并选中做装策略。前后缀正则表达式匹配原则如下
    a) 非复合词缀，数字使用".*?"匹配
    b) 复合词缀，数字使用".*?\n.*?"匹配
    c) 和复合词缀重名的单词缀，使用".*?\n(\{|$)"匹配
    d) 属性词缀(敏捷、力量、智慧)使用"\+.*?"匹配
    e) 对数字大小有要求的，如弓t1后缀的"+(24–26)% 持续伤害加成"，需专门在条件中编写正则表达式。如"\+(24|25|26)%% 持续伤害加成"
  详例可见”混沌持续弓.ini"配置
3、按Tab键主动退出
